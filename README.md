# 🎮 Gaming Store

> A simple Java-based Gaming Store application developed as an academic project using Object-Oriented Programming (OOP).

---

## 📖 About

Gaming Store is a desktop application built with Java that simulates a basic online game store. Users can sign up, log in, browse games, view game details, and complete a simple payment process. This project was created to practice Java programming, GUI development, and OOP concepts.

---

## ✨ Features

* 🔐 User Authentication (Sign In / Sign Up)
* 🏠 Welcome & Home Screen
* 🎮 Browse Game Collection
* 📄 Game Details Page
* 💳 Payment System
* ✅ Payment Success Page
* 🖼️ Image-Based Interface
* 💻 Easy-to-Use GUI

---

## 🛠️ Technologies Used

* Java
* Java Swing
* Object-Oriented Programming (OOP)

## 🚀 Getting Started

1. Clone the repository

```bash
git clone https://github.com/Tashin90/Gaming-Store.git
```

2. Open the project in any Java IDE (NetBeans, Eclipse, or IntelliJ IDEA).

3. Run `Start.java`.
